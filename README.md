o	Mô tả : 

o	Thanh công cụ có :
-	Palette: Chứa các thành phần như Button, Label, TextBox, Image,... 
-	Viewer: Khu vực thiết kế giao diện ứng dụng. 
-	Components: Hiển thị danh sách các đối tượng đã thêm vào Screen. 
-	Properties: Thay đổi thuộc tính của đối tượng (Text, màu sắc, kích thước,...).
  
o	Kéo thả và thay đổi thuộc tính
-	Chọn thành phần trong Palette rồi kéo vào Viewer. 
-	Chọn đối tượng và chỉnh sửa trong Properties. 
-	Mục đích: Thiết kế giao diện nhanh mà không cần viết mã nguồn. 

o	Block: 
-	Bản chất: MIT App Inventor tự chuyển các Block thành mã nguồn Android và biên dịch thành file APK.
-	Ưu điểm : 
+ Dễ học, không cần nhớ cú pháp lập trình. 
          + Hạn chế lỗi cú pháp. 
          + Tạo ứng dụng nhanh. 
          + Phù hợp cho người mới học lập trình.
-	Nhược điểm :
+ Khó quản lý khi ứng dụng lớn. 
+ Ít tính năng nâng cao hơn Android Studio. 
+ Hiệu năng thấp hơn Java/Kotlin. 
+ Khó tích hợp thư viện ngoài.
-	Copy paste block :
+ Backpack là công cụ lưu trữ Block tạm thời. 
+ Kéo Block vào Backpack để sao chép. 
+ Có thể kéo Block từ Backpack sang Screen khác để sử dụng lại. 
+ Giúp tiết kiệm thời gian và tái sử dụng chương trình dễ dàng.
